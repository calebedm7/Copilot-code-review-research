# Code Review Demo - C Project

## Description
A simple C project to demonstrate the Copilot code review tool with intentional code issues.

## Building the Project
```bash
make
```

## Running the Program
```bash
./demo
```

## Cleaning Build Files
```bash
make clean
```

## Intentional Issues
This project contains various code quality and security issues for demonstration purposes:
- Buffer overflow vulnerabilities
- Memory leaks
- Use of unsafe functions
- Missing input validation
- Poor error handling
- Magic numbers
- And more...

The Copilot code review tool should identify and suggest fixes for these issues.
